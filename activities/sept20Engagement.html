Engagement - Link

Explain the difference between an absolute link and a relative link. Why would you use relative links?

Absolute Link - You can put links and those links can be from your own work, or from other websites.

Relative Link - You can only add links that are from your own work.
You use relative link if you are only going to include links that are from your own work.

List 3 rules with relative links
  1. Periods (.) - It means current folder
  2. Two Periods (..) - Parent folder
  3. Slash (/) - Meaning at the beginning, start at the root.

Do links always need to be inside a NAV tag?

Links are not always required to be inside a NAV tag.
