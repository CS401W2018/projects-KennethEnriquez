Questions

1. What is the difference between block tags and inline tags.

Block tags are used for making large section of a content such has header, nav, main,
etc.Inline tags are used forbeautification of the contents inside the block tags.


2. What is the margin, border, and padding?

margin allows you to give space for the text on, top, bottom, left, and right. Border allows you to set style for a block tag or inline tag.


3. What are units of measurement you can use in CSS?

Pixels, points, inches, centimeters, and millimeters


4. When would you use a figure tag?

Figure tag is a tag use for visual contents. You use it if you
want to dispaly multiple visual contents like images or diagram.
