1. What is the difference between a block and an inline tag? Give examples of each one.

- Block tag takes width of the page and use large section. Inline tags are use for smaller inline contents. examples of block tag is div and p tags while examples of inline tags are strong, em, img.


2. What is normal flow in HTML?

- default arrangement of the tags use for making html where block elements stack vertically, and inline elements align horizontally within a line.


3. What does float do? Why do you need to clear the float?

- The float property in CSS lets elements move to the left or right side of their container, making other content flow around them. You need to clear the float so that the parent element doesn't collapse and to make sure elements after the floated one don't get wrapped around it.


4. What do absolute, fixed, and relative do?

- In CSS, absolute positioning allows an element to be placed relative to its nearest positioned ancestor or the viewport if there's none, which takes it out of the normal document flow. Fixed positioning keeps an element stuck in one spot on the browser window, so it doesn't move even when you scroll the page. Meanwhile, relative positioning shifts an element from its original position while still keeping its space in the layout, allowing it to affect other elements around it.
