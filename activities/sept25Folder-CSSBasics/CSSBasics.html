CSS Basics


Explain the difference between inline, embedded, and external CSS.

- Inline is putting the code style directly to the html. embedded is we place the style tag inside the head of the html. external css uses a stylesheet then we put its link in the html to apply the style.


Explain why you should only use external CSS.

- It will save you time and you can use it to multiple html codes.


What is a selector used for?

- selector is used for styles used in the page. You can choose where to apply it.
