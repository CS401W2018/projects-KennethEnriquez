List out the different ways to select an item.

What is the difference between an ID and a Class?

- ID uses pound sign "#", uses noun, and each tag can only have one While Class uses a period sign ".", uses adjectives, and can have more than one class.


What is the difference between a SPAN tag and a DIV tag?

- SPAN cannot handle multiple blocks unlike DIV tag that can.


What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

- They all are the same inside the webpage but DIV tag comes before the Header, Main, and Footer.


What are different ways to select a color?

- You can use style tag or use another file which format is ".css" then make modification for which tags you want to put color.
