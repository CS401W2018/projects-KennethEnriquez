Definitions of terms

Internet

- Internet is a network that allows computers around the world to connect.

Web

- Web are public websites that a person can access using their phones and computers through the internet.

How are the Internet and Web different?

- Internet allows you to connect with other computers while web contains the content that you access through the internet.

ASCII and Unicode

- ASCII or American Standard Code for Information Interchange assigns numbers to a special character such as letters, numbers, and special characters like punction marks and it has a total of 128 characters, while Unicode is similar to ASCII but it has 149,000 characters including characters of different languages.

What is UTF-8

- It is similar to ASCII but the only difference is that it includes special characters like chinese and Arabic.

URL

- URL or Uniform Resource Locator is used to locate or identify a specific resource in the internet.

Domain Name

- Domain name gives a specific website a unique name which you use to easily search or access the desired website.

HTTP

- HTTP or Hypertext Transfer Protocol are use by the user and the server for communication such as transfering files or datas through the internet.

HTTPS

- It is the same with HTTP but it is more secure since it encrypts the connection of the client and the server.

IP Address

- It shows all devices that are connected either to the internet or local network through a sets of numbers.

DNS

- DNS or Domain Name System is a system that converts a domain name into an IP address for computers to be read.

W3C Recommendations

- W3C Recommendations or World Wide Web Consortium Recommendations is a guideline for web development to make everything unified.

XML, XHTML, HTML, HTML5

XML or Extensible Markup Language provides rules to define a data.
XHTML or Extersible HyperText Markup Language is basically anHTML but follows the rules of XML
HTML or Hypertext Markup Language is a code to make a web page.
